Insert your Yeelight Bulb IP addresses in properties.txt, each separated by a single comma and without any spaces.
Invalid IP's will be ignored.
Additionally, before running, you must set the following environment variables:
SPOTIPY_CLIENT_ID = client_id
SPOTIPY_CLIENT_SECRET = client_secret
SPOTIPY_REDIRECT_URI = https://google.com/

If you do not have these, head to https://developer.spotify.com/dashboard/login to generate your client id and client secret.
